# BadgeEditorQt Glossary

このファイルは、バッジ制作まわりで出てきやすい用語を、プロジェクト内の呼び方に揃えるための辞典です。

## 使い分けの方針

- `バッジ` は総称として使う
- `缶バッジ` は紙を透明フィルムで覆う丸型の量産品を指す
- `ピンバッジ` / `ピンズ` は針や留め具で衣服につける金属系アイテムを指す
- UI 表示は日本語を優先し、内部データ名はできるだけ短くする
- 業者や地域で呼び方が揺れるものは、備考に残す

## 用語

| 用語 | 意味 | このプロジェクトでの扱い |
| --- | --- | --- |
| 缶バッジ | 紙の印刷物を透明フィルムで覆い、裏に留め具が付く丸型バッジ | 量産の基本単位として扱う |
| ピンバッジ | 針やピンで留める徽章の総称 | 社章や記念品も含む広い意味で使う |
| ピンズ | 日本ではピンバッジの意味で使われることが多い呼び方 | `ピンバッジ` と同義扱いにしてよい |
| ラペルピン | 襟元につける小型のピン系アクセサリ | 高級感のある小型徽章の文脈で使う |
| 社章 | 会社や団体の所属を示すバッジ | テンプレートや量産用途で想定する |
| 裏紙 / バックペーパー | バッジ裏面の印刷やメーカー表示 | 将来の版面情報として扱える余地を残す |
| 留め具 | バッジを衣服に固定する部品の総称 | UI では最もわかりやすい言葉を使う |
| バタフライクラッチ | 一般的な差し込み式の留め具 | `通常バック` と並べて表示してもよい |
| ロックピン | 抜けにくい固定式の留め具 | 量産・展示向けの選択肢として扱う |
| マグネットバック | 磁石で留める方式 | 衣服を傷めたくない用途向け |
| ソフトエナメル | 金属の仕切りが立ち、色面が少し凹む製法 | 立体感のある表現として扱う |
| ハードエナメル | 表面を研磨して平滑に仕上げる製法 | 高級感のある表現として扱う |
| 塗り足し | 仕上がりより外側まで伸ばす余白 | 印刷安全のため必須の概念 |
| 安全圏 | 重要な要素を置いてよい内側の範囲 | 文字や顔を守るための基準にする |
| トンボ | 印刷位置合わせのためのマーク | 出力機能で扱う予定の概念 |
| テンプレート | 複製して量産するための元データ | レイアウト自動配置の入力として扱う |
| レイアウト | 用紙上への配置結果 | Layout 画面の主役 |
| 版面 | 実際に印刷される範囲 | 将来的な出力データで使う |

## 実装メモ

- `BadgeData` は 1 個のバッジを表す
- `LayerData` は重ねる画像や装飾を表す
- `PaperConfig` は用紙設定を表す
- `DocumentData` はプロジェクト全体の保存単位

## UI での表記候補

- `Designer` -> `編集`
- `Layout` -> `配置確認`
- `Badge` -> `バッジ`
- `Layer` -> `レイヤー`
- `Paper` -> `用紙`
- `Export` -> `出力`

## 迷ったときの判断

- 業界用語より、まずユーザーに伝わる言葉を優先する
- ただし保存形式や内部データ名は一貫性を優先する
- 言葉が揺れるときは、UI と内部で名前を分ける
